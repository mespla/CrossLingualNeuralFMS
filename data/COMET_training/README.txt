This folder contains two files: a compressed package, that contains the training and dev data used to train our COMET model, and a configuration file referenceless_metric.yaml to train the specific model. The COMET version used to build the model was the one available at https://github.com/Unbabel/COMET, with the specific commit c9e1818583c1677328e3d1b649bb6c9c6.
